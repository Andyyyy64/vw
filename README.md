# vw

Have you ever wondered why your workspace is so dirty? No, you haven't.

## Overview

vw visualizes your workspace in 3D to make it easier to.... hmm, see?

## Installation

```bash
npm install -g vw
```

## Usage

Navigate to your project directory and run:

```bash
vw open
```

This will open your default browser at `http://localhost:5554` (port may vary if 5554 is in use).

## Development Setup

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Build the project:
    ```bash
    npm run build
    ```
4.  Link the binary locally for testing:
    ```bash
    npm link
    ```
